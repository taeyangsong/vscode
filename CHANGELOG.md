# Changelog — Priority Guard

변경 이력은 날짜 역순으로 기록합니다.

---

## [0.2.0] — 2026-04-03

### Added
- **드래그 앤 드롭 순위 변경** — 할일 항목을 드래그해서 순서 재배치, `POST /tasks/reorder`
- **인라인 편집** — 할일 제목 더블클릭으로 바로 수정
- **AI 끼워넣기 자동 밀기** — AI가 추천한 순위로 추가 시 기존 항목 순위 자동 +1, `POST /tasks/insert`
- **어제 미완료 가져오기** — 헤더 버튼으로 어제 미완료 항목을 오늘 목록에 복사, `POST /tasks/carry-over`
- **주간 완료율 통계** — 최근 7일 막대 차트, 오늘은 초록색, `GET /tasks/stats`
- **Toast 알림** — 주요 액션 완료 시 하단 피드백 메시지

### Changed
- UI 전면 개선: 드래그 핸들, 통계 바, 헤더 레이아웃
- AI 판단 결과에 "목록 맨 뒤에 추가" 버튼 추가 (`later` 케이스)

## [0.1.0] — 2026-04-03

### Added
- **할일 CRUD** — 우선순위와 함께 오늘 할일 등록 / 완료 / 삭제
- **새 요청 AI 판단** — Claude Haiku가 🔴끼워넣기 / 🔵나중에 / ⚪오늘 안 해도 됨 판단
- **하루 요약 AI** — 완료·미완료 기반 오늘 평가 + 내일 조언
- FastAPI 백엔드, SQLite DB, Vanilla JS 프론트엔드
- `.env` 기반 API 키 관리
